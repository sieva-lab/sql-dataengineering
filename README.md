# SQL Learning Lab: DuckDB & Chinook

A hands-on environment for mastering advanced SQL using **DuckDB**. 

## Concepts Covered

* Relational Joins: Connecting Artists, Albums, and Tracks.

* Window Functions:
ROW_NUMBER() for deterministic Top N lists.
RANK() vs DENSE_RANK() for handling ties.
SUM() OVER() for running totals.

* Tie-breaking Logic: 
Using secondary sort keys (e.g., ORDER BY count DESC, id ASC) for consistent results.
